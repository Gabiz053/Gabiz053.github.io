Eventos – Eliminarse de evento

UMLUseCase
Youtify :: Plantillas :: Eventos – Eliminarse de evento

Description

Nombre Eliminarse de evento. Descripción El usuario cancela la compra de la entrada a un evento y se le elimina de la lista de personas que acuden al evento. Actores primarios Usuario. Actores secundarios ----- Evento de activación El caso de uso comienza cuando el Usuario selecciona la opción eliminarse de un evento. Precondiciones ---- Flujo principal 1. El sistema pide al Usuario que introduzca el artista que hostea el evento. 2. El Usuario introduce el artista que hostea el evento. 3. El sistema pide al Usuario que introduzca el evento del artista seleccionado del que quiere eliminarse. 4. El Usuario introduce el evento del que quiere eliminarse. 5. El sistema comprueba que el evento no ha empezado o finalizado. 6. El sistema busca los datos del Usuario en la lista del evento seleccionado. 7. Include (Reembolso de entrada). 8. Include (Eliminarse de lista del evento). Postcondiciones Se le muestra al usuario un mensaje confirmando que se le ha eliminado correctamente del evento y el dinero ha sido reembolsado. Flujos alternativos 5.a El evento ha empezado o finalizado. 5.a.1 El sistema muestra un mensaje de error de que el evento ya ha empezado o finalizado. 6.a El usuario no está en la lista. 6.a.1 El sistema muestra un mensaje de error de que no se encuentra en la lista del evento. En cualquier momento el usuario cancela la opción de eliminarse de un evento. 1. El sistema sale de la opción de eliminarse de un evento
