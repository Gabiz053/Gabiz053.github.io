Artista – Publicar Canción

UMLUseCase
Youtify :: Plantillas :: Artista – Publicar Canción

Description

Nombre Publicar canción. Descripción El artista pone todos los datos y publica una canción. Actores primarios Artista. Actores secundarios ---- Evento de activación El caso de uso comienza cuando el Artista selecciona la opción de “publicar canción”. Precondiciones ---- Flujo principal 1. El sistema pregunta al usuario los detalles de la canción a añadir (nombre, descripción, duración). 2. El cliente introduce toda la información requerida. 3. El sistema valida la información recibida. 4. El sistema guarda la información recibida de manera temporal. 5. El sistema pregunta al usuario por el fichero de la canción y una foto para la carátula. 6. El usuario introduce los archivos requeridos. 7. El sistema valida los archivos subidos. 8. El sistema guarda los archivos de manera temporal. Extension point: añadir canción a disco. 9. El sistema añade la canción al sistema de manera permanente. 10. El sistema elimina la información temporal almacenada. Postcondiciones El usuario recibe una confirmación de haber creado la canción correctamente. Flujos alternativos 3.a El nombre introducido no es válido. a1. El sistema preguntará por el nombre hasta 3 veces. 3.b La descripción introducida no es válida. b1. El sistema preguntará por la descripción hasta 3 veces. 3.c La duración introducida no es válida. c1. El sistema preguntará por la duración hasta 3 veces. 7.a El formato de la canción subida no es válido. a1. El sistema pide al usuario subir un archivo distinto. 7.b El formato de la carátula subida no es válido. b1. El sistema pide al usuario subir un archivo distinto. 9.a La base de datos no se encuentra disponible. a1. El sistema mantiene la información temporal y reintenta acceder a la base de datos pasados 1 minuto, 5 minutos y 1 hora.
